## 1. 项目概述

### 1.1 项目背景

随着城市化进程加速和机动车保有量持续增长，交通排放已成为城市空气污染的重要来源。为了改善空气质量，各地政府陆续实施了一系列机动车管控政策（如限行、限购、鼓励新能源车等）。然而，如何科学评估这些政策的环境效益，量化其对空气质量改善的贡献，成为政策制定者和研究者面临的重要课题。

传统的评估方法往往依赖于单一数据源或简化的线性模型，难以捕捉复杂的非线性关系和时空动态变化。因此，构建一个基于多源数据融合与深度学习模型的机动车管控政策效益评估系统具有重要的现实意义。

### 1.2 项目目标

本系统的核心目标是：

1. **多源数据整合**：整合交通流量、机动车类型、空气质量监测、气象观测等多源数据，构建高时空分辨率的机动车排放清单。

2. **智能预测分析**：基于深度学习模型对污染物浓度的时空分布进行预测，提高预测精度。

3. **政策效益量化**：通过特征贡献度分析，定量揭示不同道路类型、时段及政策情景下机动车排放对空气质量的影响机制。

4. **科学决策支持**：为机动车管控政策的制定、实施与优化提供科学、可靠的决策依据。

### 1.3 项目范围

本系统主要包括以下功能模块：

- **数据管理模块**：多源数据的采集、存储、清洗与预处理
- **排放清单模块**：机动车排放清单的构建与动态更新
- **模型管理模块**：深度学习模型的训练、评估与部署
- **政策评估模块**：政策情景模拟与效益分析
- **可视化展示模块**：评估结果的查询、展示与对比分析
- **系统管理模块**：用户管理、权限控制、系统配置等

| 维度         | 要求                                       |

```
原始数据采集 → 数据清洗 → 数据融合 → 特征工程 → 排放清单构建 → 模型输入
```

## 7. 系统架构概述

### 7.1 总体架构

系统采用前后端分离的三层架构：

```
┌─────────────────────────────────────────────────────────┐
│                        表现层                            │
│                    （Vue.js 前端）                      │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐  │
│  │数据展示  │ │政策评估  │ │可视化    │ │系统管理  │  │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘  │
└─────────────────────────────────────────────────────────┘
                            ↓ REST API
┌─────────────────────────────────────────────────────────┐
│                        业务层                            │
│                  （Spring Boot 后端）                    │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐  │
│  │数据管理  │ │模型服务  │ │评估引擎  │ │用户管理  │  │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘  │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                        数据层                            │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐  │
│  │ MySQL    │ │  Redis   │ │ InfluxDB │ │文件存储  │  │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘  │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                    外部服务层                            │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐                │
│  │数据采集  │ │深度学习  │ │任务调度  │                │
│  └──────────┘ └──────────┘ └──────────┘                │
└─────────────────────────────────────────────────────────┘
```
